# Betfair Analysis Tools

Folder pentru analize live ale API-ului Betfair și identificare oportunități.

## Scripts disponibile:

1. `live_market_scanner.py` - Scanează piețe live pentru oportunități
2. `arbitrage_detector.py` - Detectează oportunități de arbitraj
3. `odds_movement_tracker.py` - Monitorizează mișcări de cote
4. `liquidity_analyzer.py` - Analizează lichiditatea piețelor
5. `market_inefficiencies.py` - Identifică ineficiențe de piață

## Utilizare:

```bash
cd backend/analysis
python live_market_scanner.py
```
